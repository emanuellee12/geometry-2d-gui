# 2D Geometry Application with GUI

An interactive application for drawing and managing 2D geometric shapes, built in Java.

---

## Features

- Draw and manage 2D shapes: circle, triangle, rectangle, and polygon
- Full graphical user interface (GUI)
- OOP design: interfaces, inheritance, polymorphism, and deep copy
- Full JUnit unit tests
- Javadoc documentation

---

## Technologies

`Java` `OOP` `JUnit` `Javadoc`
